🎮 RPG Python Game

A simple text-based RPG game built with Python.
The player can fight monsters, gain XP, level up, and collect loot.

⚔️ Features
Turn-based combat system
Player stats (HP, Attack, Level, XP)
Monster battles
Level up system
Random loot drops
Simple inventory system
🧠 How it works
You fight monsters by choosing actions from a menu
Defeating monsters gives XP
Enough XP → level up
Monsters drop random items
You can view stats and inventory anytime
🕹️ Controls
1 - Attack Monster
2 - Heal
3 - Show Stats
4 - Show Inventory
0 - Exit Game
🚀 How to run
Make sure you have Python installed
Run the file:
python main.py
